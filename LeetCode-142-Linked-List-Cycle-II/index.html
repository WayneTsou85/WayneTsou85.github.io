142. Linked List Cycle II

Given the head of a linked list, return the node where the cycle
begins. If there is no cycle, return null.

There is a cycle in a linked list if there is some node in the list that can be reached again by continuously following the next pointer. Internally, pos is used to denote the index of the node that tail’s next pointer is connected to (0-indexed). It is -1 if there is no cycle. Note that pos is not passed as a parameter.

Do not modify the linked list.

解題

快慢指針

  • 慢指針 → Slow，快指針 → Fast
  • Slow一次一步，Fast一次兩步
  • Slow移動的距離 * 2 = Fast移動的距離 (2∗Slow=Fast2*Slow=Fast2∗Slow=Fast)
  • 當Fast與Slow交會就表示這個LinkedList有Cycle

如何找出Cycle的起點?

  • 假設：
    • xxx = 起始點到Cycle的起點距離
    • yyy = Cycle的起點到第一次交會點距離
    • zzz = 第一次交會點到Cycle的起點距離
  • 套用快慢指針的情境
    2∗Slow=Fast2*Slow=Fast2∗Slow=Fast
    2∗(x+y)=x+y+z+y2*(x+y)=x+y+z+y2∗(x+y)=x+y+z+y
    x=zx=zx=z
  • 將慢指針退回到起點，而快指針每次改移動一步，再找一次交會點就是Cycle的起點
